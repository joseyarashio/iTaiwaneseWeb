使用遠端 Scripting

遠端  Scripting 讓您能夠在 Web 應用程式中將兩個 Scripting 平台結合在一個網頁中。您可以建立客戶端 Script 來控制應用程式的使用者介面，而且如果您正在使用 Microsoft Internet Explorer 4.0，還可以善加利用動態 HTML (Dynamic HTML，DHTML) 的強效功能。同時，您可以在伺服程式上執行 Script 來執行伺服端工作，包括資料庫查詢、中階商務邏輯等。因為當客戶端網頁仍在使用時，遠端 scripting 是在伺服端上執行，您的 scripts 會被大幅簡化，而且應用程式可以出現更豐富、更能回應使用者的介面。
