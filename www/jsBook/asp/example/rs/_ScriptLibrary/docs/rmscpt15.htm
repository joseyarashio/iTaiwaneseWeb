wait 方法

停止目前正在執行的客戶端 script，直到指定呼叫物件的非同步遠端 scripting 呼叫完成為止。

語法

co.wait()

參數

co
執行遠端 scripting 呼叫所產生的呼叫物件。

請注意

呼叫此方法會將非同步遠端 scripting 呼叫轉換為同步遠端 scripting 呼叫。如果遠端 scripting 呼叫已經完成，此方法會立即傳回。如果您在客戶端 script 中碰到某個點需要來自遠端 scripting 呼叫的結果時，便可以使用此呼叫。

範例

下面的 script 會以非同步方式呼叫 EmpLists 網頁上的 GetNames 方法。啟動呼叫之後，螢幕上會顯示訊息，然後等待其結果。

<SCRIPT LANGUAGE="JavaScript" FOR="btnReport" EVENT="onclick">
    empLookup = RSGetASPObject("../mypages/EmpLists.asp");
    co = empLookup.GetNames(showResults);
    txt1.value = "Getting employee names ..."
    co.wait()    // suspends client processing
</SCRIPT>
