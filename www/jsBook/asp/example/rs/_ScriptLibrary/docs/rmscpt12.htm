RSEnableRemoteScripting 方法

在客戶端啟動遠端 scripting。

語法

RSEnableRemoteScripting(codebase)

參數

codebase
遠端 scripting proxy applet (Rsproxy.class) 所在位置的路徑，是一個選擇性的規格。如果您並未指定 codebase 參數，RSEnableRemoteScripting 方法會在目前伺服器或專案的根目錄下之 _ScriptLibrary 資料夾中尋找 Rsproxy.class 程式碼。

請注意

客戶端網頁必須在嘗試進行遠端 scripting 呼叫之前先呼叫此方法，以啟動遠端呼叫能力。在客戶端網頁的生命週期中，此方法只須被呼叫一次。
