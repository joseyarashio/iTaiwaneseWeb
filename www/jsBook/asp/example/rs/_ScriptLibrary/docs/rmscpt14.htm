RSGetASPObject 方法

建立設定為可接收遠端 scripting 呼叫的 ASP 網頁之物件引用。

語法

objName = RSGetASPObject(url)

參數

objName
您可以用來啟動 url所指定網頁中的方法之物件變數。

url
ASP 網頁的路徑和檔案名稱，此網頁包含您想要呼叫的方法，其格式為 //server/.../filename.asp。

請注意

藉由建立 ASP 網頁的物件引用，您就可以使用標準的 object.method 語法來呼叫該物件的方法。如果您並未建立 ASP 物件的物件引用，還是可以呼叫此物件的方法，但是必須使用比較複雜的 RSExecute 方法。

例如，下列範例在第一行建立一個物件變數 empLookup ，它會引用 ASP 網頁 EmpLists.asp。下一行則呼叫 ASP 網頁的 GetNames 方法︰

<SCRIPT LANGUAGE="JavaScript">
empLookup = RSGetASPObject("../mypages/EmpLists.asp");
co = empLookup.GetNames();
</SCRIPT>
