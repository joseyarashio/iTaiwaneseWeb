將 ASP 網頁當作物件引用

如果您希望能夠更方便地使用遠端 scripting 呼叫 ASP 網頁上的方法，可以在客戶端 script 中建立一個對包含伺服端方法之網頁的物件引用。這樣，您就可以使用標準的 object.method( ) 語法，呼叫您在伺服端網頁上公佈的方法。

若要將 ASP 網頁當作物件引用

  • 在客戶端 script 中，呼叫 RSGetASPObject 方法，把您想引用之 ASP 網頁的 URL 和名稱當作物件傳送。此 RSGetASPObject 方法會傳回一個物件引用。請使用下列語法︰
    ASPobj = RSGetASPObject(url)

    其中︰

    • ASPobj   表示以 RSGetASPObject 方法所建立的物件。此後，您就可以使用這個物件來引用 ASPName.asp 中指定的網頁。

    • url   為伺服端網頁的 URL，此網頁上有您想要呼叫的方法。

例如，假設在伺服器上有一個名為 RSMath.asp 的網頁。您就可以用下面這種以 JavaScript 寫成的陳述式來建立物件引用︰

rsMath = RSGetASPObject("../myPages/RSMath.asp")

或是用 VBScript 寫成的陳述式︰

set rsMath = RSGetASPObject("../myPages/RSMath.asp")

如果該網頁中包含一個名為 Add的方法，您可以使用下列以 JavaScript 寫成的語法來呼叫此方法︰

co = rsMath.Add(number1, number2)

或是用 VBScript 的語法︰

set co = rsMath.Add(number1, number2)

關於呼叫遠端 scripting 方法的相關資訊，請參閱同步呼叫遠端 Scripting 方法和非同步呼叫遠端 Scripting 方法。
