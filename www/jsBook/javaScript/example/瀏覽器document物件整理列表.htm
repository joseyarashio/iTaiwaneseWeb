瀏覽器Document物件整理列表

Document物件的性質
Properties Description 中文說明
activeElement Retrieves the object that has the focus when the parent document has focus. 確認取得焦點的元件
aLinkColor Sets or retrieves the color of all active links in the document. 即正要連結的超連結的顏色,所謂正要連結指的是滑鼠指標在連結按下的剎那
bgColor Sets or retrieves the background color behind the document object. 背影顏色
body Specifies the beginning and end of the document body.  
cookie Sets or retrieves the string value of a cookie. 存取網頁內的cookie物件
defaultCharset Sets or retrieves the default character set of the document. 設定或傳回文件的預設字元集
designMode Sets or retrieves whether the document can be edited. 切換文件的瀏覽及編輯模式
dir Sets or retrieves the reading order of the document object.  
documentElement Retrieves a reference to the root node of the document. 傳回文件的根元件
domain Sets or retrieves the security domain of the document. 設定或傳回文件的安全網域
expando Sets or retrieves whether arbitrary variables can be created within an document object.  
fgColor Sets or retrieves the foreground (text) color of the document. 前景(即一般文字)顏色
fileCreatedDate Retrieves the date the file was created. 傳回檔案的建立日期
fileModifiedDate Retrieves the date the file was last modified. 傳回檔案的修改日期
fileSize Retrieves the file size. 傳回檔案的大小
lastModified Retrieves the date the page was last modified, if the page supplies one. 即文件最近一次修改的日期與時間
linkColor Sets or retrieves the color of the document links. 尚未被瀏覽過的超連結的顏色
location Contains information about the current URL. 即文件的URL位址
media Sets or retrieves the type of media used to render a document.  
parentWindow Retrieves a reference to the container object of the window. 傳回包含此網頁文件的上層視窗
protocol Sets or retrieves the protocol portion of a URL. 設定或傳回目前文件URL使用的通訊協定
readyState Retrieves the current state of the document object. 設定正在下載的物件的狀態
referrer Retrieves the URL of the location that referred the user to the current page. 傳回前一個瀏覽網頁的URL
selection Represents the active selection, which is a highlighted block of text,and/or other elements in the document on which a user or a script can carry out some action.參照Selection元件的資訊,讓Script能控制它
title Contains the title of the document.  
uniqueID Retrieves an autogenerated, unique identifier for the document object. 傳回一個文件物件自行產生的唯一ID
URL Sets or retrieves the URL for the current document. 設定或傳回目前文件的URL位置
URLUnencoded Retrieves the URL for the document, stripped of any character encoding.  
vlinkColor Sets or retrieves the color of the links that the user has visited. 已經瀏覽過的超連結的顏色
XMLDocument Retrieves a reference to the XML Document Object Model (DOM) exposed by the object.  
XSLDocument Retrieves a reference to the top-level node of the XSL document.  

Document物件的方法

Method Description 中文說明
attachEvent Binds the specified function to an event,so that the function gets called whenever the event fires on the object.  
clear Clears the current document. 清除選擇區域內的文件內容
clearAttributes Removes all attributes and values from the document object. 清除件中所有的屬性及值
close Closes an output stream and forces the sent data to display. 關閉文件
createElement Creates an instance of the element for the specified tag. 建立特定標籤的物件實體
createEventObject Generates an event object for passing event context information when using the fireEvent method.  
createStyleSheet Creates a style sheet for the document. 建立一個樣式表
createTextNode Creates a text string from the specified value. 建立一個特定值的純文字字串
detachEvent Unbinds the specified function from the event,so that the function stops receiving notifications when the event fires. 
elementFromPoint Returns the element for the specified iX and iY coordinates. 傳回目前位於X,Y座標上的物件
execCommand Executes a command on the current document, current selection, or the given range. 在特定的選取範圍上執行某個命令
focus Causes the element to receive the focus and executes the code specified by the onfocus event.  
getElementById Returns a reference to the first object with the specified value of the ID attribute. 利用ID屬性的值,傳回集合中的元件
getElementsByName Retrieves a collection of objects based on the value of the NAME attribute. 利用NAME屬性的值,傳回集合中的元件
getElementsByTagName Retrieves a collection of objects based on the specified element name. 利用標籤名稱,傳回集合中的元件
hasFocus Retrieves the value indicating whether the document object currently has focus. 
mergeAttributes Copies all read/write attributes to the specified element. 複製可讀寫屬性到特定元件上
open Opens a document to collect the output of write or writeln methods. 開啟一份新文件,以顯示所有write和writeln方法的輸出結果
queryCommandEnabled Returns a Boolean value that indicates whether a specified command can be successfully executed using execCommand, given the current state of the document.表示是否可使用execCommand設定的命令
queryCommandIndeterm Returns a Boolean value that indicates whether the specified command is in the indeterminate state. 表示設定的命令是否處於未定狀態
queryCommandState Returns a Boolean value that indicates the current state of the command. 傳回命令的目前狀態
queryCommandSupported Returns a Boolean value that indicates whether the current command is supported on the current range. 表示在目前的區域上,是否支援目前的命令
queryCommandText   傳回命令有關的字串
queryCommandValue Returns the current value of the document, range, or current selection for the given command. 傳回命令的值
recalc Recalculates all dynamic properties in the current document. 重新計算文件中的動態樣式性質
releaseCapture Removes mouse capture from the document object in the current document. 移除Mouse擷取功能
setActive Sets the document object as the current object without setting focus to the object. 
write Writes one or more HTML expressions to a document in the specified window. 字串顯示到瀏覽器視窗
writeln Writes one or more HTML expressions, followed by a carriage return,to a document in the specified window. 和write類似,但會在行末多加一個換行字元

Document物件的事件

Event Description 中文說明
onactivate Fires when the document object is set as the activeElement. 
onbeforecut Fires on the source object before the selection is deleted from the document.在剪下選取的物件前所發生的事件
onbeforedeactivate Fires immediately before the activeElement is changed from the current document object to another object in the parent document.  
onbeforeeditfocus Fires before a control enters a UI activated state. 在編輯區域取得焦點前所發生的事件
onbeforepaste Fires on the target object before the selection is pasted from the system clipboard to the document. 在貼上選取的物件前所發生的事件
onclick Fires when the user clicks the left mouse button on the document object. 在物件上按下滑鼠左鈕
oncontextmenu Fires when the user clicks the right mouse button in the client area,opening the context menu. 在物件上按下滑鼠右鈕
oncontrolselect Fires when the user is about to make a control selection of an document object. 
oncut Fires on the source element when the document object or selection is removed from the document and added to the system clipboard. 當所剪下的物件送至剪貼簿時所發生的事件
ondblclick Fires when the user double-clicks the document object. 在文件上雙按滑鼠鈕
ondeactivate Fires when the activeElement is changed from the current document object to another object in the parent document. 
ondrag Fires on the source object continuously during a drag operation. 當使用者持續拉曳物件時
ondragend Fires on the source object when the user releases the mouse at the close of a drag operation. 當使用者結束以拉曳的方式移動選取的文字區域或是物件時
ondragenter Fires on the target element when the user drags the document object to a valid drop target. 當使用者將物件拉曳進入可放置的區域時,被拉曳的物件就會發生此事件
ondragleave Fires on the target object when the user moves the mouse out of a valid drop target during a drag operation. 當使用者拉曳物件時,如果將物件拉曳離開一個有的拉曳目標區時,被拉曳的物件就會發生此事件
ondragover Fires on the target element continuously while the user drags the document object over a valid drop target. 當使用者在可以拉曳的區域持續拉曳物件時
ondragstart Fires on the source object when the user starts to drag a text selection or selected object. 當使用者開始以拉曳的方式,移動取的文字區域或物件時
ondrop Fires on the target object when the mouse button is released during a drag-and-drop operation. 在拉曳過程中,使用者若放開滑鼠鈕,則被拉曳物件就會發生此事件
onhelp Fires when the user presses the F1 key while the browser is the active window. 當使用者按下系統定義的輔助說明鍵,如<F1>時
onkeydown Fires when the user presses a key. 當使用者按下按鍵時
onkeypress Fires when the user presses an alphanumeric key. 當使用者按下任意鍵時,會產生onKeydown事件,著再產生onKeyPress事,若使用者按住按鍵不放,則會產生連續的onKeyPress事件
onkeyup Fires when the user releases a key. 當使用者放開按鍵時
onmousedown Fires when the user clicks the document object with either mouse button. 當使用者按下滑鼠按鈕時
onmousemove Fires when the user moves the mouse over the document object. 當使用者移動滑鼠指標時
onmouseout Fires when the user moves the mouse pointer outside the boundaries of the document object. 當滑鼠指標離開物件時
onmouseover Fires when the user moves the mouse pointer into the document object. 當滑鼠指標移到物件上方時
onmouseup Fires when the user releases a mouse button while the mouse is over the document object. 當使用者放開滑鼠鈕時
onpaste Fires on the target object when the user pastes data, transferring the data from the system clipboard to the document. 當取或剪下的物件貼到目的文件時
onpropertychange Fires when a property changes on the document object. 當文件樣式改變時
onreadystatechange Fires when the state of document object has changed. 當文件的readyState性質改變時
onresizeend Fires when the user finishes changing the dimensions of the document object in a control selection. 
onresizestart Fires when the user begins to change the dimensions of the document object in a control selection. 
onselectionchange Fires whenever the selection state of a document changes.  
onstop Fires when the user clicks the Stop button or leaves the Web page. 當使用者按下瀏覽器的停止鈕或文件從目前視窗中移除時

Document物件的集合

Collections Description 中文說明
all Returns a reference to the collection of elements contained by the document object. 文件中所有標籤和物件的集合
anchors Retrieves a collection of all A objects that have a name and/or id property.Objects in this collection are in HTML source order. 文件中所有anchor物件的集合
applets Retrieves a collection of all APPLET objects in the document. 文件中所有Applet物件的集合
childNodes Retrieves a collection of HTML Elements and TextNode objects that are direct descendants of the specified object. 文件中某元件的所有子元件集合
children Retrieves a collection of DHTML Objects that are direct descendants of the document object. 文件中所有子元件集合
embeds Retrieves a collection of all EMBED objects in the document. 文件中所有嵌入物件的集合
forms Retrieves a collection, in source order, of all FORM objects in the document. 文件中所有表單物件的集合
frames Retrieves a collection of all window objects defined by the given document or defined by the document associated with the given window. 文件中所有窗格物件的集合
images Retrieves a collection, in source order, of IMG objects in the document. 文件中所有圖像物件的集合
links Retrieves a collection of all A objects that specify the HREF property and all AREA objects in the document. 文件中所有連結物件的集合
namespaces Retrieves a collection of namespace objects.  
scripts Retrieves a collection of all SCRIPT objects in the document. 文件中所有Script程式的集合
styleSheets Retrieves a collection of styleSheet objects representing the style sheets that correspond to each instance of a LINK or STYLE object in the document. 文件中所有樣式性質物件的集合
