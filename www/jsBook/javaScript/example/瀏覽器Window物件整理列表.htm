window物件的性質
Properties Description 中文說明
clientInFormation Contains information about the Web browser. 存取navigator物件.
clipboardData Provides access to predefined clipboard formats for use in editing operations. 定義編輯時,剪貼簿上的資料格式.
closed Retrieves whether the referenced window is closed. 指示參考窗格是否已經關閉.
dafaultStatus Sets or retrieves the default message displayed in the status bar at the bottom of the window. 存取瀏覽器狀態列的預設訊息文字.
dialogArguments Retrieves the variable or array of variables passed into the modaldialog window. 將已傳遞給Modal交談窗的參數以陣列方式傳回
dialogHeight Sets or retrieves the height of the modal dialog window. 設定或傳回交談窗的高度
dialogLeft Sets or retrieves the left coordinate of the modal dialog window. 設定或傳回交談窗的X座標.
dialogTop Sets or retrieves the top coordinate of the modal dialog window. 設定或傳回交談窗的Y座標.
dialogWidth Sets or retrieves the width of the modal dialog window. 設定或傳回交談窗的寬度.
document Represents the HTML document in a given browser window. 代表特定瀏覽視窗中的HTML文件
event Represents the state of an event, such as the element in which the event occurred, the state of the keyboard keys, the location of the mouse, and the state of the mouse buttons. 代表事件的狀態,例如:案件事件發生在哪一個元件上,滑鼠位置,滑鼠鈕按鈕的狀態,及鍵盤的狀態等.
external Allows access to an additional object model provided by host applications of the MicrosoftR Internet Explorer browser components. 允許使用IE瀏覽器所提供的附加物件
frameElement Retrieves the FRAME or IFRAME object that is hosting the window in the parent document.  
history Contains information about the URLs visited by the client. 傳回目前瀏覽過的URL資訊.
length Sets or retrieves the number of objects in a collection. 傳回集合裡的元件數目
location Contains information about the current URL. 網頁的URL位址
name Sets or retrieves the window or frame name. 視窗或窗格的名稱.
navigator Contains information about the Web browser. 包含瀏覽器應用程式的資訊
offscreenBuffering Sets or retrieves whether objects are drawn offscreen before beingmade visible to the user. 設定顯示物件時,是否使用背景緩衝處理
opener Sets or retrieves a reference to the window that created the current window. 設定或傳回開啟目前視窗的參考視窗資訊.
parent Retrieves the parent of the window in the object hierarchy. 傳回物件的上層視窗
returnValue Sets or retrieves the value returned from the modal dialog window. 設定或傳回modal交談視窗的傳回值
screen Contains information about the client's screen and rendering capabilities. 包含客戶端螢幕及相關顯示能力的資訊
screenLeft Retrieves the x-coordinate of the upper left-hand corner of the browser's client area, relative to the upper left-hand corner of the screen. 傳回視窗相對於整個螢幕的X座標
screenTop Retrieves the y-coordinate of the top corner of the browser's client area, relative to the top corner of the screen. 傳回視窗相對於整個螢幕的Y座標
self Retrieves a reference to the current window or frame. 及目前視窗或窗格的參考值
status Sets or retrieves the message in the status bar at the bottom of the window. 設定或傳回狀態列的顯示文字
top Retrieves the topmost ancestor window. 傳回最上層的視窗物件

window物件的方法

Methods Description 中文說明
alert Displays a dialog box containing an application-defined message. 開啟顯示訊息交談窗.
attachEvent Binds the specified function to an event, so that the function gets called whenever the event fires on the object.  
blur Causes the window object to lose focus and fires the onblur event. 使元件失去焦點,引發onBlur事件.
clearInterval Cancels the interval previously started using the setInterval method. 取消clearInterval方法設定的計時器.
clearTimeout Cancels a time-out that was set with the setTimeout method. 取消clearTimeout方法設定的計時器.
close Closes the current browser window or HTML Application (HTA). 關閉目前的瀏覽器視窗或是HTA視窗.
confirm Displays a confirmation dialog box that contains an optional message as well as OK and Cancel buttons. 開啟確認訊息交談窗.
createPopup Creates a popup window.  
detachEventUnbinds the specified function from the event, so that the function stops receiving notifications when the event fires. 
execScriptExecutes the specified script in the provided language.執行Script程式,預設是Jscript
focusCauses the element to receive the focus and executes the code specified by the onfocus event.使元件取得焦點,並引發onFocus事件.
moveByMoves the screen position of the window by the specified iX and iY offset values.將視窗相對於目前的位置移動X和Y個像素.
moveToMoves the screen position of the upper-left corner of the window to the specified iX and iY position.將視窗位置移至X,Y座標上.
navigateLoads the specified URL to the current window. 載入一份文件.
openOpens a new window and loads the document specified by a given URL, or opens a blank document if a URL is not provided. 開啟新視窗.
printPrints the document associated with the window.  
promptDisplays a dialog box that prompts the user with a message and an input field. 開啟prompt輸入交談窗.
resizeByChanges the current size of the window by the specified x- and y-offset.將視窗縮小X和Y個像素.
resizeToSets the size of the window to the specified width and height values.以特定的寬度和高度來設定視窗的大小.
scrollCauses the window to scroll to the specified x- and y-offset at the upper-left corner of the window.  
scrollByCauses the window to scroll relative to the current scrolled position by the specified x- and y-pixel offset. 
scrollToScrolls the window to the specified x- and y-offset.  
setActiveSets the window object as the current object without setting focus to the object. 
setIntervalEvaluates an expression each time a specified number of milliseconds has elapsed.設定在固定的時間間距,重複執行某個函式或程序.
setTimeoutEvaluates an expression after a specified number of milliseconds has elapsed.設定等待某一段時間後,開始執行某個函式或程序.
showHelpDisplays a Help file. This method can be used with Microsoft HTML Help.顯示一個HTML help的交談窗.
showModalDialogCreates a modal dialog box that displays the specified HTML document.顯示一個新的交談窗.
showModelessDialogCreates a modeless dialog box that displays the specified HTML document. 

window物件的事件

events Description 中文說明
onActivateFires when the window object is set as the activeElement. 
onAfterPrint Fires on the window object immediately after its associated document prints or previews for printing. 文件列印之後所發生的事件.
onBeforeDeactivate Fires immediately before the activeElement is changed from the current window object to another object in the parent document.  
onBeforePrint Fires on the window object before its associated document prints or previews for printing. 列印文件之前所發生的事件.
onBeforeUnload Fires prior to a page being unloaded. 當網頁要被移除前發生.
onBlur Fires when the window object loses the input focus. 發生在視窗失去焦點時,也就是作用視窗被切換為背景視窗時.
onControlSelect Fires when the user is about to make a control selection of an window object.  
onDeactivate Fires when the activeElement is changed from the current window object to another object in the parent document.  
onError Fires when an error occurs during window object loading. 載入網頁文件發生錯誤時發生.
onFocus Fires when the window object receives the focus. 發生在視窗取得焦點時,也就是背景視窗被切換為作用視窗時.
onHelp Fires when the user presses the F1 key while the browser is the active window. 使用者按下F1件時發生.
onLoad Fires immediately after the browser loads the window object. 網頁載入瀏覽器視窗時發生.
onResize Fires when the size of the window object is about to change. 發生在使用者調整視窗大小時.
onResizeEnd Fires when the user finishes changing the dimensions of the window object in a control selection. 發生在使用者完成調整視窗大小時.
onResizeStart Fires when the user begins to change the dimensions of the window object in a control selection. 發生在使用者要調整視窗大小之前.
onUnload Fires immediately before the window object is unloaded. 網頁文件從瀏覽視窗移除時發生,例如使用者預連往其他網站或是關閉瀏覽器視窗時./td>

window物件的集合

Collections Description 中文說明

window.history物件

Properties Description 中文說明
lengthRetrieves the number of elements in the History list.傳回紀錄清單裡的紀錄數目.
Methods Description 中文說明
back Loads a previous URL from the History list. 載入紀錄清單裡之前瀏覽過的第n筆URL紀錄,這個方法相當於按[上一頁]鈕.
forward Loads the next URL from the History list. 載入紀錄清單裡之前瀏覽過的第n筆URL紀錄,這個方法相當於按[下一頁]鈕.
go Loads a URL from the History list. 載入當對於目前瀏覽位置之第n筆URL.

window.navigator物件

Properties Description 中文說明
appCodeName Retrieves the code name of the browser. 傳回瀏覽器應用程式的代號名稱,不過無論你是用IE或是Netscape Navigator,appCodeName的傳回值一率都是Mozilla.
appMinorVersion Retrieves the application's minor version value. 傳回瀏覽器應用程式的版本編號中的小數部分.
appName Retrieves the name of the browser. 傳回瀏覽器應用程式的名稱.
appVersion Retrieves the platform and version of the browser. 傳回瀏覽器應用程式的版本.
browserLanguage Retrieves the current browser language. 傳回瀏覽器使用的語言.
cookieEnabled Retrieves whether client-side cookies are enabled in the browser. 表示瀏覽器是否啟動接受來自網路上的Cookie;傳回值為True(自動接收)或False(不自動接收).
cpuClass Retrieves a string denoting the CPU class. 傳回使用者電腦的CPU等級.
onLine Retrieves whether the system is in global offline mode. 偵測使用者的瀏覽器是否設定為離線瀏覽.
platform Retrieves the name of the user's operating system. 傳回使用者使用的作業系統名稱.
systemLanguage Retrieves the default language used by the operating system. 傳回使用者電腦使用的語言.
userAgent Retrieves a string equivalent to the HTTP user-agent request header. 傳回瀏覽器應用程式的代號和版本,也就是appCodeName和appVersion的結合.
userLanguage Retrieves the current user language. 傳回使用者目前使用的語言.
userProfile Provides methods that allow a script to request read access to and perform read actions on a user's profile information. 提供Script讀取使用者資料的方法.
Methods Description 中文說明
javaEnabled Returns whether Java is enabled. 表示瀏覽器是否啟動執行java的功能,傳回值為True(啟動)或False(關閉).
taintEnabled Returns whether data tainting is enabled. 表示是否能和Netscape Navigator相容,傳回值為True(相容)或False(不相容).
Collections Description 中文說明
plugins Retrieves a collection of all EMBED objects within the document. 傳回所有利用<EMBED>標籤所插入的物件集合.

window.location物件

Properties Description 中文說明
hash Sets or retrieves the subsection of the href property that follows the number sign (#). 設定或傳回URL裡跟在#之後的字串.
host Sets or retrieves the hostname and port number of the location or URL. 設定或傳回主機名稱與連結阜編號.
hostname Sets or retrieves the host name part of the location or URL. 設定或傳回名稱.
href Sets or retrieves the entire URL as a string. 設定或傳回完整的URL.
pathname Sets or retrieves the file name or path specified by the location object. 設定或傳回目錄路徑,也就是主機名稱之後的字串.
port Sets or retrieves the port number associated with a URL. 設定或傳回連結阜編號.
protocol Sets or retrieves the protocol portion of a URL. 設定或傳回協定名稱.
search Sets or retrieves the substring of the href property that follows the question mark. 設定或傳回搜尋字串,也就是?之後的字串.
Methods Description 中文說明
assign Loads a new HTML document. 載入另一個網頁.
reload Reloads the current page. 重新載入目前的網頁.
replace Replaces the current document by loading the document at the specified URL. 連結另一個新的網頁.

window.event物件

Properties Description 中文說明
altKey Sets or retrieves the state of the ALT key. 傳回Alt鍵的狀態;傳回值為True(Alt鍵被按下)或False(Alt鍵沒有被按下)
altLeft Sets or retrieves the state of the left ALT key. 傳回left Alt鍵的狀態;傳回值為True(left Alt鍵被按下)或False(left Alt鍵沒有被按下)
behaviorCookie Retrieves a cookie identifying the rendering behavior on which this event was fired.  
behaviorPart Retrieves a value that identifies the part of a rendering behavior on which this event was fired.  
button Sets or retrieves the mouse button pressed by the user. 傳回是哪一個滑鼠鈕被按下.
cancelBubble Sets or retrieves whether the current event should bubble up the hierarchy of event handlers. 設定或傳回目前發生的事件是否要繼續在瀏覽器物件的層級架構中向上沸升.
clientX Sets or retrieves the x-coordinate of the mouse pointer's position relative to the client area of the window, excluding window decorations and scroll bars. 傳回滑鼠的X座標.
clientY Sets or retrieves the y-coordinate of the mouse pointer's position relative to the client area of the window, excluding window decorations and scroll bars. 傳回滑鼠的Y座標.
contentOverflow Retrieves a value that indicates whether the document contains additional content after processing the current LayoutRect object.  
ctrlKey Sets or retrieves the state of the CTRL key.傳回Ctrl鍵的狀態;傳回值為True(Ctrl鍵被按下)或False(Ctrl鍵沒有被按下)
ctrlLeft Sets or retrieves the state of the left CTRL key.傳回left Ctrl鍵的狀態;傳回值為True(left Ctrl鍵被按下)或False(left Ctrl鍵沒有被按下)
dataFld Sets or retrieves the data column affected by the oncellchange event. 傳回受OnCellChange事件影響的資料欄.
dataTransfer Provides access to predefined clipboard formats for use in drag-and-drop operations. 提供可利用滑鼠拉曳,存取剪貼簿格式上預先設定格式的方式.
fromElement Sets or retrieves the object from which the mouse pointer is exiting during the onmouseover and onmouseout events. 傳回當onMouseover 或onMouseOut事件發生時,滑鼠游標正離開的哪個元件.
keyCode Sets or retrieves the Unicode key code associated with the key that caused the event. 傳回或設定按鍵的ASCII值.來引發其他事件.
nextPage Retrieves the position of the next page within a print template.  
offsetX Sets or retrieves the x-coordinate of the mouse pointer's position relative to the object firing the event. 傳回事件發生時,滑鼠相當於引發事件之物鍵的水平距離.
offsetY Sets or retrieves the y-coordinate of the mouse pointer's position relative to the object firing the event. 傳回事件發生時,滑鼠相當於引發事件之物鍵的垂直距離.
propertyName Sets or retrieves the name of the property that changes on the event object. 傳回目標物件因事件發生而改變的樣式名稱.
qualifier Sets or retrieves the name of the data member provided by a data source object. 傳回DSO所提供的資料欄位名稱.
reason Sets or retrieves the result of the data transfer for a data source object. 了解DSO傳輸資料是否成功.
recordset Sets or retrieves from a data source object a reference to the default record set. 傳回一個參考DSO預設資料集的recordest物件.
repeat Retrieves whether the onkeydown event is being repeated. 傳回事件是否一直重複發生.
returnValue Sets or retrieves the return value from the event. 設定或傳回該事件的傳回值.
saveType Retrieves the clipboard type when oncontentsave fires.  
screenX Retrieves the x-coordinate of the mouse pointer's position relative to the user's screen. 傳回事件發生時,滑鼠相對於螢幕左上角的x座標.
screenY Retrieves the y-coordinate of the mouse pointer's position relative to the user's screen. 傳回事件發生時,滑鼠相對於螢幕左上角的y座標.
shiftKey Sets or retrieves the state of the SHIFT key. 傳回Shift鍵的狀態.傳回值為True(Shift鍵被按下)或False(Shift鍵沒有被按下).
shiftLeft Sets or retrieves the state of the left SHIFT key. 傳回left Shift鍵的狀態.傳回值為True(left Shift鍵被按下)或False(left Shift鍵沒有被按下).
srcElement Sets or retrieves the object that fired the event. 傳回第一個引發事件的元件.
srcFilter Sets or retrieves the filter object that caused the onfilterchange event to fire. 傳回使元件發生onfilterchange事件的濾鏡.
srcUrn Retrieves the Uniform Resource Name (URN) of the behavior that fired the event. 傳回引發事件的Behavior之Universal Resource Name.
toElement Sets or retrieves a reference to the object toward which the user is moving the mouse pointer. 在onMouseOver或onMoveOut事件裡,滑鼠游標正前往的元件.
type Retrieves the event name from the event object. 以字串傳回事件名稱,但去除開頭的"on".
x Sets or retrieves the x-coordinate, in pixels, of the mouse pointer's position relative to the parent element. 傳回滑鼠指標相對於父元件的x座標.
y Sets or retrieves the y-coordinate, in pixels, of the mouse pointer's position relative to the parent element 傳回滑鼠指標相對於父元件的y座標.
