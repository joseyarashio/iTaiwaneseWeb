JavaScript 程式設計與應用

張智星

本光碟為「JavaScript 程式設計與應用」（第二版）之範例光碟，您可以進行下列選項：

  1. 測試本書之範例程式碼：

    建議您先將本光碟之範例目錄 jsBook 拷貝至硬碟 d:\jsBook（或其他適合的路徑），再設定 IIS 的虛擬目錄（名稱也是 jsBook）來指到此本機硬碟中的範例目錄，就可以直接經由下列網頁的 localhost 連結方式來執行這些範例檔案（但 WSH 檔案仍然必須下載後在 DOS 視窗執行）。有關於如何設定 IIS 的虛擬目錄，可參閱本書第二篇的第一章。

    • 第一篇：用於網頁客戶端
    • 第二篇：用於伺服器ASP環境
    • 第三篇：用於單機WSH環境（此篇之範例並無法直接點選執行，您必須先下載檔案後，再使用 cscript fileName.js 或是 cscript fileName.wsf 的方式在 DOS 視窗下執行。詳情請參閱本書第三篇。）

    （本書範例均根據 IE 測試，如果您不是使用 IE，可能無法正確呈現網頁效果唷！）

  2. 本書之範例列表：

    下列網頁中的連結都是指到硬碟中的本機檔案。

    • 第一篇：用於網頁客戶端
    • 第二篇：用於伺服器ASP環境
    • 第三篇：用於單機WSH環境

  3. 檢視光碟內容：

    假設本光碟放在 E 槽，則您可以直接點選此連結或在瀏覽器輸入 "file:///E:/" 來檢視本光碟之內容。若不是，請直接開啟「檔案總管」，以右鍵點選光碟機圖示，並選取「開啟」。
