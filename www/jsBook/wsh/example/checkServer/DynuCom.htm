〔範例程式／其他 ASP 元件〕作者：小莊　　日期：2001.03.03

元件名稱：dynuCOM tools

出品公司：dynu http://www.dynu.com

　　Powerful Tools: datatime, DNS, Email, Encrypt, Exec, File, FTP, HTTP, Ping, Pop, String, Upload, Wait, Whois.

　

〔說明〕

　　一、dynuCOM 元件的安裝方法

　　將本範例解壓縮之後，把 source 子目錄底下一個主要的 DLL 檔案複製到系統目錄。這一個檔案名叫：dynuCOM.dll

　　windows 2000 的系統目錄是：c:\winnt\system32
　　windows 98 的系統目錄是：c:\windows\system

　　進 DOS 模式，到系統目錄下執行：regsvr32 dynucom.dll 即可。

　　本範例所附的是免費試用版。

　　二、元件方法、屬性

set dynu = server.createobject("Dynu.Exec") exec 元件
dynu.execute("ipconfig")
　
set myping = server.createobject("Dynu.Ping") ping 元件
dynu.ping("127.0.0.1")
　
set dynu = server.createobject("Dynu.DateTime") 日期時間元件
dynu.Format(...) 格式化日期輸出
dynu.StartClock() 開始計時
dynu.StopClock() 停止計時
dynu.ElapsedTime(...) 格式化計時輸出
　
set dynu = Server.CreateObject("Dynu.Encrypt") 加解密元件
dynu.encrypt(txt,key) 加密資料
dynu.decrypt(txt,key) 解密資料
　
set mywait = server.createobject("Dynu.Wait") 等待元件
mywait.Wait(n) 暫停 n 秒
　
set myemail = Server.CreateObject("Dynu.Email") 郵件 SMTP 元件
myemail.Host 代送郵件的主機
myemail.AddAddress 收件人
myemail.AddCC 副件收件人
myemail.AddBcc
myemail.Subject 郵件主旨
myemail.Body 郵件內容
myemail.AddAttachment 附加檔案
myemail.Send() 開始傳送
　
set myemail = Server.CreateObject("Dynu.POP") 郵件 POP3 元件
mypop.Connect("host","username", "password") 連線至郵件主機
mypop.GetStatistics() 取得郵件資訊
mypop.NumberOfMails 新郵件數目
mypop.Retrieve(n) 下載第 n 封信件
mypop.GetHeader() 檔頭
mypop.GetFrom() 寄信者
mypop.GetDate() 日期
mypop.GetReplyTo() 回覆
mypop.GetSubject() 主題
mypop.GetBody() 內容
　
set myfile = Server.CreateObject("Dynu.FileUtil") 檔案元件
myfile.GetAttributes(filename) 取得檔案屬性
myfile.Open(filename) 開啟檔案
myfile.Read(n) 讀取 n 個字元
myfile.Write(text) 寫入字串
myfile.GetModifiedTime(filename) 取得檔案最近修改時間
myfile.GetSize(filename) 取得檔案大小
myfile.Close() 關閉檔案
　
set myftp = Server.CreateObject("Dynu.FTP") FTP 元件
myftp.ExpressGetFile("ftp.dynu.com", "dynuftp.zip") 下載檔案
myftp.connect("ftp.dynu.com", 21, "anonymous", "s@s.com") 進行連線
myftp.GetCurrentDirectory() 取得目前所在目錄
myftp.ExecuteCommand("NOOP") 送出 noop 指令
myftp.ExecuteCommand("LIST") 送出 list 指令
myftp.ExecuteCommand("NLST") 送出 nlst 指令
myftp.ExecuteCommand("SYST") 送出 syst 指令
myftp.Close() 斷線
　
set myhttp = server.CreateObject("Dynu.HTTP") HTTP 元件
myhttp.SetHeader "user-agent","DynuHTTP"
myhttp.SetURL "http://www.dynu.com"
myhttp.GetHeader()
myhttp.GetURL()
myhttp.Reset()
myhttp.SetQueryString "t=55"
myhttp.SetFormData "formdata", "John Doe"
myhttp.PostURL()
　
set mystring = server.createobject("Dynu.StringUtil") 字串處理元件
mystring.Combine(x,y)
mystring.Insert(x, 5, "!")
　
set myup = server.CreateObject("Dynu.Upload") 上傳元件
myup.SavePath 存檔目錄
myup.MaximumSize = 5000000 檔案大小上限
myup.OverwriteFiles = TRUE 是否覆蓋
myup.Upload() 開始上傳
　
set mywhois = Server.CreateObject("Dynu.Whois") 查詢元件
mywhois.Whois("microsoft.com")
mywhois.RemoveCopyright = FALSE
mywhois.WhoisServerAddress = "whois.tonic.to"
mywhois.WhoisServerPort = 43
mywhois.IsAvailable("memory.to")
mywhois.WhoisHtml("memory.to")
　
set mydns = Server.CreateObject("Dynu.DNS") DNS 元件
mydns.Lookup("dynu.com")
mydns.ReverseLookup("198.144.10.35")
mydns.GetLocalIP()
mydns.GetLocalHost()
mydns.IsIPValid("198.144.10.35")
mydns.Ping("dynu.com")
　
set mycc = Server.CreateObject("Dynu.CreditCard") 信用卡交易元件
mycc.CheckCard("5329053144000902","M", 10, 2001)
